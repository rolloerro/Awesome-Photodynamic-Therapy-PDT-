# 🤝 Contributing to Awesome Photodynamic Therapy (PDT)

First of all — **thank you for contributing** to the Photodynamic Therapy community!  
This repository exists to collect **reliable, practical, and clinically relevant** materials related to PDT.

We welcome contributions from:
- clinicians
- researchers
- engineers
- students
- medical startups
- open-source contributors

---

## 🎯 What You Can Contribute

You can contribute in many ways:

### 📚 Documentation & Knowledge
- PDT protocols
- clinical guidelines
- safety notes
- educational materials
- translations

### 🧮 Tools & Calculators
- PDT dose calculators
- light power / exposure tools
- validation formulas
- scripts (Python, JS, etc.)

### 🤖 Bots & Services
- Telegram / VK bots
- clinical assistants
- form generators

### 🧾 Templates & Forms
- informed consent
- clinical reports
- protocols
- emergency forms
 
### 🔗 Curated Links
- scientific papers
- trusted articles

- official guidelines

## 🚀 Quick Start (for New Contributors)

1. **Fork** this repository
2. **Create a new branch**:
   ```bash
   git checkout -b feature/my-contribution
Add your content (see structure below)

Commit with a clear message:

git commit -m "Add PDT calculator for dose calculation"
Push and open a Pull Request

📂 Repository Structure
Please place your contribution in the appropriate section:

/docs        → documentation, protocols, explanations
/tools       → calculators, scripts, utilities
/bots        → chatbots and automation tools
/forms       → PDFs, templates, clinical forms
/resources   → links, references, datasets
If you are unsure — open an Issue first, we will help.

🧠 Medical Responsibility Disclaimer
⚠️ Important

All materials in this repository are:

for educational and informational purposes

NOT a replacement for professional medical judgment

should always be validated against local clinical protocols

Contributors are responsible for:

accuracy of information

proper sourcing

clear warnings when required

✅ Pull Request Guidelines
Please ensure that your PR:

is focused on one clear change

includes a short explanation of why it’s useful

does not break existing structure

respects medical accuracy

Screenshots, formulas, or references are highly appreciated.

🧪 Review & Merge Process
Maintainers review PRs manually

Feedback may be requested

Once approved, PRs are merged into main

Be patient — quality matters more than speed 🧘‍♂️

💬 Communication
Use Issues for ideas and questions

Use Discussions for concepts and architecture

Be respectful and constructive

We are building this together.
